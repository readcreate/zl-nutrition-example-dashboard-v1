Aucun enfant ne passe entre les mailles du filet.
No child falls through the cracks.

Ce tableau de bord transforme les données CollaborateCare/CommCare en intelligence actionnable — alertes en temps réel, suivi par patient, et visibilité pour chaque rôle de l'équipe PIH.

This dashboard transforms CommCare data into actionable intelligence — real-time alerts, per-patient tracking, and role-specific visibility for every PIH team member.

🎯
Le problème central
The core problem

Les données existent dans CommCare — mais personne ne les surveille de manière systématique. Un enfant peut manquer deux rendez-vous sans qu'aucune alerte ne soit envoyée. Ce tableau de bord change cela.

The data exists in CommCare — but nobody watches it systematically. A child can miss two appointments without any alert being sent. This dashboard changes that.

Système d'alertes / Alert system
📋
Résumé quotidien
Daily digest
Chaque matin, chaque ASC reçoit un résumé de ses patients à visiter. Via WhatsApp.
🚨
Alerte urgente
Urgent alert
Pour événements critiques seulement: oedème, perte de poids, absence prolongée. Via WhatsApp.
📊
Rapport hebdomadaire
Weekly report
Pour les superviseurs et managers chaque vendredi. Via email.
🔴
Escalade superviseur
Supervisor escalation
Si une alerte urgente n'est pas acquittée sous 24h, le superviseur est notifié. WhatsApp + Email.
Caractéristiques des données fictives / Dummy data characteristics
Patients actifs: 14 patients fictifs dans 3 programmes (USN, PTA, PNS) / 14 synthetic active patients across 3 programs
Patients sortis: 15 patients fictifs avec résultats / 15 discharged patients with outcomes
Sites: 6 vrais sites PIH Haïti — Mirebalais, Hinche, Lascahobas, Boucan Carré, Sapaterre, Pignon / 6 real PIH Haiti sites
ASC: 7 agents de santé fictifs, un par site / 7 fictional CHWs, one per site
Période: Admissions oct–jan 2025–2026 / Admissions Oct–Jan 2025–2026
Langue: Données en français, interface bilingue FR/EN
Seuil visite USN: Alerte si >7 jours sans visite / Alert if >7 days without visit
Seuil visite PTA/PNS: Alerte si >14 jours sans visite / Alert if >14 days without visit
Score de risque: Basé sur retard de visite, tendance MUAC, gain de poids (<5g/kg/jour), oedème / Based on visit delay, MUAC trend, weight gain, oedema
⚠ Validation clinique requise: Les seuils utilisés sont basés sur les protocoles WHO/Sphere SAM/MAM. Ils doivent être validés par l'équipe clinique PIH avant toute mise en production. / Thresholds based on WHO/Sphere SAM/MAM protocols — must be validated by PIH clinical team before production use.
Assignation ASC: Attribution fictive pour la démo. En production, cette donnée doit provenir de CommCare. / Fictional for demo. In production, must come from CommCare.
Glossaire / Glossary
SAM — Malnutrition aiguë sévère / Severe Acute Malnutrition. MUAC <115mm ou P/T <70%.
MAM — Malnutrition aiguë modérée / Moderate Acute Malnutrition. MUAC 115–125mm ou P/T 70–79%.
USN — Unité de Stabilisation Nutritionnelle (hospitalisation SAM).
PTA — Programme Thérapeutique Ambulatoire (SAM ambulatoire).
PNS — Programme Nutritionnel Supplémentaire (MAM/préventif).
MUAC / PB — Périmètre brachial / Mid-upper arm circumference.
P/T — Poids-pour-taille / Weight-for-height.
ASC — Agent de Santé Communautaire / Community Health Worker (CHW).
